# To Do App API

REST API для управления задачами, разработанный на Spring Boot. Проект реализует функционал CRUD и содержит интеграционные тесты, а также CI/CD пайплайн.

---

## Описание

Приложение позволяет:

- создавать задачи
- получать список всех задач
- обновлять задачи
- удалять задачи

Данные хранятся в оперативной памяти (in-memory список), база данных не используется.

---

## Технологический стек

- Java 17+
- Spring Boot
- Spring Web
- JUnit
- CI/CD (GitHub Actions / pipeline)

---

## REST API

### Создать задачу

**POST** `/tasks`

```json
{
  "id": 1,
  "title": "Купить молоко",
  "createdAt": "2026-04-17",
  "done": false
}
```

---

### Получить все задачи

**GET** `/tasks`

---

### Обновить задачу

**PUT** `/tasks/{id}`

```json
{
  "id": 1,
  "title": "Купить хлеб",
  "createdAt": "2026-04-17",
  "done": true
}
```

---

### Удалить задачу

**DELETE** `/tasks/{id}`

---

## Обработка ошибок

| Ошибка            | HTTP статус   |
| ----------------- | ------------- |
| TaskNotFound      | 404 Not Found |
| TaskAlreadyExists | 409 Conflict  |

---

## Тестирование

В проекте используются интеграционные тесты на MockMvc.

Запуск тестов:

```bash
mvn test
```

Тесты проверяют:

- создание задач
- получение списка задач
- обновление задач
- удаление задач
- корректность ответа API

---

## CI/CD

Проект включает CI/CD pipeline, который:

- собирает проект
- запускает тесты
- проверяет корректность сборки
